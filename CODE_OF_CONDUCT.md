# Code of Conduct

## Our Standard

This project aims to be useful, technically rigorous, and respectful to contributors of all backgrounds. Participation should stay constructive, specific, and focused on improving the code, documentation, and user experience.

## Expected Behavior

- Be respectful and precise in technical discussions.
- Assume good intent, but challenge ideas with evidence.
- Keep feedback actionable and repository-focused.
- Help newcomers understand context instead of gatekeeping it.

## Unacceptable Behavior

- Personal attacks, insults, or harassment
- Discriminatory language or exclusionary behavior
- Repeated off-topic disruption in issues, PRs, or discussions
- Publishing private information without consent

## Enforcement

Project maintainers may edit, remove, or close comments, issues, or pull requests that violate this code of conduct. Repeated or severe violations may result in a ban from participation.

## Reporting

For conduct-related concerns, open a private security-style report if needed or contact the maintainer directly via the repository contact links.
